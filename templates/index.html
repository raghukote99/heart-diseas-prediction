Check your heart health

This demo project uses a machine learning model trained on the Cleveland Heart Disease dataset to estimate your risk level. It is made for educational purposes and is not a medical diagnosis.

What you’ll need
  • Age, sex and chest pain type
  • Blood pressure and cholesterol values
  • Resting ECG and maximum heart rate
  • Other clinical indicators (oldpeak, slope, etc.)

Your inputs are stored in a secured database so an admin can review model performance later. No personal name or contact details are collected in this demo.

How it works

1
Enter your clinical values

Fill in the form with your health parameters like BP, cholesterol, heart rate, and more.

2
Model predicts risk

A trained KNN model processes your inputs and estimates the likelihood of heart disease.

3
Download a PDF report

View the result along with model confidence and download a neatly formatted PDF summary.

About this project

This single-page Flask web app is created as an academic project to demonstrate how machine learning can be integrated with a modern UI and a backend database.
